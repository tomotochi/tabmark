# Changelog

## [0.1.2] - 2025-12-07

### Added

- `validateTabmarkStructure()` method to validate Tabmark structure
- `TabmarkValidationResult` interface for validation results
- Validation checks for H1 (sheet name), H2 (numeric row numbers), and H3 (column names)

## [0.1.0] - 2025-12-06

### Added

- Initial release of `tabmark-core`.
- Hierarchical Markdown parsing.
- Grid data conversion.
- HTML and Markdown escaping.

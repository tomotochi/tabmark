# Tabmark for VS Code

他の言語で読む: [English](./README.md) | 日本語

Tabmark は、**Markdown の階層構造**（`#`、`##`、`###` などの見出し）をスプレッドシートのようなデータグリッドとして再解釈する VS Code 拡張機能です。通常の Markdown テーブル記法（`|---|---|`）のエディタ**ではありません**。

## Tabmark とは？

Tabmark は Markdown の見出し階層を使って表形式データを表現します：
- `#` (H1) = シート名
- `##` (H2) = 行番号
- `###` (H3) = 列名
- その下のコンテンツ = セルの値

**なぜ `.table.md`？** 中身は普通の Markdown ですが、Tabmark の構造規約に従っていることを示し、DataGrid エディタで開くべきファイルであることを明示するために `.table.md` というファイル拡張子（サフィックス）を使用しています。

## なぜこのアプローチ？

この設計は、2つの目標を組み合わせるために作られました：
- **AI フレンドリー**: Markdown は LLM に非常に読みやすく、AI 支援ワークフローに最適
- **人間フレンドリー**: DataGrid インターフェースでスプレッドシート編集の利便性を提供

Markdown の階層構造を活用することで、機械可読性と視覚的なグリッド編集の両方を実現しています。

## 機能

- **DataGrid インターフェース**: 階層的 Markdown を使い慣れたスプレッドシートとして編集
- **リッチなセルコンテンツ**: セル内にリスト、コードブロック、複数段落などの複雑な Markdown を含められます
- **CSV 統合**: CSV ファイルからのインポートと CSV ファイルへのエクスポートが可能
- **モダンな技術スタック**: Svelte 5 と Vite で構築され、高いパフォーマンスを実現

## 使い方

1. VS Code で `*.table.md` ファイルを開きます
2. ファイルは自動的に Tabmark DataGrid エディタで開かれます
3. コマンドパレットから「Export to CSV」または「Import from CSV」コマンドを使用してデータを変換します
